FEATURES

ETH-LESS TRANSACTIONS

Marmo allows Ethereum addresses to complete transactions without requiring funds in Ether (ETH) or any token.

ZERO GAS MATH

The Relayer handles Gas Price and Gas Limits calculations automatically, simplifying interactions with the Ethereum network.

NONCE-FLEXIBLE CODE

The SDK allows to write transactions with any number of dependencies (including zero), making it possible to execute them in any order or without any in particular.

EXPIRABLE TRANSACTIONS

Marmo enables the creation of transactions with expirations dates, which will always get executed within a predefined time frame.
